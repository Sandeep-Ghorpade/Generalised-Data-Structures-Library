# DSA Library in C++  
A complete collection of generic Data Structure and Algorithm implementations using C++ templates.  
This project contains multiple core data structures, searching techniques, and sorting algorithms written in a clean and reusable format.

---

##   Features

###   Linear Data Structures
- Queue (Template-based)
- Stack (Template-based)
- Singly Linear Linked List (SLL)
- Doubly Linear Linked List (DLL)

###   Circular Linked Lists
- Singly Circular Linked List (SCLL)
- Doubly Circular Linked List (DCLL)

###   Tree Data Structure
- Binary Search Tree (BST)
  - Insert
  - Inorder / Preorder / Postorder Traversal
  - Search
  - Count Nodes / Leaf Nodes / Parent Nodes

###   Sorting Algorithms
- Bubble Sort
- Efficient Bubble Sort
- Selection Sort
- Insertion Sort

###   Searching Algorithms
- Linear Search
- Bi-directional Search
- Binary Search (Ascending)
- Binary Search (Descending)
- Efficient Binary Search
